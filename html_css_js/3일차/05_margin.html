        margin : 요소의 바깥쪽 영역
        단위 : px , 0px은 px 생략가능

        크기 종류)
        margin-top: 50px;
        margin-right: 30px;
        margin-bottom: 70px;
        margin-left: 40px; 

        속기법 : 시계방향 (생략시 메커니즘)
        margin : top right bottom left
        margin : top right bottom       >>> 반대편 right값이 동일하게 적용됨
        margin : top right                 >>> 반대편 값들이 동일하게 적용됨
        margin : top                        >>> 모든값에 적용됨
