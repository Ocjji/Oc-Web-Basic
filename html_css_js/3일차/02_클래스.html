        * 클래스 선택자 *
        별명/별칭
        한 페이지에 여러번 사용가능
        점수로는 10점

        형식) 정의할 시,
        <태그 class="클래스명">
        
        'style.css' 에서는 .클래스명

        클래스명 규칙
        1. 영문자, 숫자, _ , - 혼용 가능
        2. 첫글자는 반드시 숫자 제외
           주로 영문자 소문자
        3. 대소문자 구별됨
        4. 특수문자/한글 사용불가
        5. 두 단어가 합쳐질 경우,
           1) 단어-단어 : dog-cat
           2) 단어_단어 : dog_cat
           3) 단어단어첫글자대문자 :
              dogCat
        6. 주로 영역( div 집 ) 클래스를
           작성하는 것이 좋다

        클래스 GUIDE를 작성하여 규칙을 정한 뒤 작업한다
        예)
        클래스 : dog-cat
        아이디 : dogCat
