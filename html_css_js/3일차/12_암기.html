################### 암기 ###################
        
        1. 블럭요소

                div, h1~h6, p, ul, li, ol, dl, dd

            1) 너비는 부모크기만큼 인식한다(100%) : 한줄
            2) 너비/높이 지정가능
            3) padding, margin 모두 가능

        2. 인라인요소

                a, span, em, b, i, strong

            1) 자신의 내용 크기만큼(컨텐츠만큼) 인식 / 옆으로 나열
            2) 너비, 높이 사용불가
            3) margin, padding - 좌우만 가능
               단, 상하가능함. 다른 컨텐츠에 영향을 줄 수 없다

        3. 인라인블럭요소

                img, dt, button, input

            1) 자신의 내용만큼만 인식(옆으로 나열) - inline 성격
            2) 너비/높이 가능 - block 성격
            3) margin / padding 가능 - block 성격
            4) 옆 사이 공백이 존재
               해결 : 부모의 요소에 font-size:0처리
               이후 자신의 요소에 다시 font-size 지정
