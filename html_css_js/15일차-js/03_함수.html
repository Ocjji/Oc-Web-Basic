함수란?

        자주 사용하는 명령들(실행문, 수식, 제어문, 내용 등)을 담아 두었다가
        필요시 호출하여 사용할 수 있다

        장점 : 코드 줄이기, 코드 재사용

        형식 1 )
        
        할일 함수명 () {
            실행문
        }

        function 함수명 () { : 함수명은 변수명과 규칙 동일
            실행문
        } : 정의시

        함수명(); - 호출


        형식 2 )
        function 함수명 ( 매개변수, 매개변수 ) {
            실행문
        }
        함수명 ( 값, 값 );


        형식 3 )
        function 함수명 ( 매개변수, 매개변수 ) {
            실행문
            return 반환값
        }
        let 변수명 = 함수명 ( 값, 값 );
