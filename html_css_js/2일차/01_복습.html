1. hN ( N = 1 ~ 6 ) : 제목
        2. p : 단락, 문단
        3. br : 강제 개행( 빈태그 ) * 빈태그는 혼자 사용 가능함
        4. ol : 순서 '있는' 목록
                자식 li만 가능
                li 자식은 모든 태그 가능
        5. ul : 순서 '없는' 목록
                자식 li만 가능
                li 자식은 모든 태그 가능
                비슷한 성격의 나열
                동일한 형태(성격) 나열
                예) 배너, 갤러리, 메뉴
        6. dl : 용어 정의
                자식 dt, dd만 가능
                dt안에는 인라인 형식만
                dt 용어
                dd 설명
                dd 자식으로 모든 태그 가능함
        7. imng : 이미지
                  속성
                  src : 이미지주소
                  alt : 대체 텍스트
                  width : 너비
                  height : 높이
                  title : 도움말
        8. a : 링크
               속성
               href : url 주소
               target : _blank ( 새탭 )
               title : 도움말
        9. div : 공간분할
                 의미없음 ( 남용하지말자 )
                 css 화면결과 구조
                 그룹핑
        10. strong : 강 강조
        11. em : 약 강조
        12. span : 의미없는 영역 ( 작은영역 )
        13. b : 의미없는 영역
        14. i : 폰트 아이콘
