        //////////////////////////////////////
        선택자 {속성:값; 속성:값; }
        선택자 {속성:값; 속성:값 } <- ';'생략 문장
        맨 끝 속성은 ';' 생략가능
        태그, 아이디, 클래스 등

        color : 글자색
        background-color : 배경색
        font-size: xxpx : 글자크기
        width : 너비px
        height : 높이px

        ##############  우선순위  ##############
        *; (모든) : 0 점    
        태그 : 1 점
        # (아이디) : 100 점
        . (클래스) : 10 점
        : (가상선택자) : 10 점

        #wrap : 100
        #wrap #header : 200
        .gnb li : 11
        .gnb li a : 12
        ul li a : 3
        a:hover : 11
