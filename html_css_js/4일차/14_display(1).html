        ######################## display ########################

        

        
        실제값의 변경 X , 화면에 보이는 성격 변경

        - inline : 블럭 to 인라인 - 거의 사용 안함
        - block : 인라인 to 블럭 - 가장 많이 사용
                화면에 보이기(js)
                none : 화면에서 사라지기
        - inline-block : 10% 정도 사용
                박스 옆으로 나열
                작은 영역의 나열(상단유틸메뉴 등)

                float : 15%
                주(flex) : 80%
