# 비밀번호 변환 생성기 #

        예외값은 처리하지 않습니다. (9자리 이상 등)


        비밀번호 입력 : 0~8자리
        
        입력된 비밀번호 왼쪽에 남은 자리수 크기만큼

        역순으로 숫자가 채워집니다.
                

        예 1)
        입력값 : 87 (8자리에서 6자리가 모자라기 때문에)

        변환된 비밀번호 : 654321 + 87
        
        출력값 : 65432187
        
        예 2)
        입력값 : 8794 (8자리에서 4자리가 모자라기 때문에)

        변환된 비밀번호 : 4321 + 8794

        출력값 : 43218794
